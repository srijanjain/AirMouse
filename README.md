# AirMouse
The Virtual Mouse

## Hardware Required
- RaspberryPi3
- Any Compatible Camera

## Software Required
- Python 2.x
- OpenCV
- Raspbian OS


## Instructions:
- Make 2 green(any green) coloured finger caps using paper, plastic etc, as shown in the poster. 
- Place them on your index and thumb finger.
- Now run the application: pinch to left click. and move hand round to move.
